Environment

  • Random Access Memory: 16 GB
  • Programming Language: Python 2.7
  • Libraries:
    • OpenCV3 – image IO, bilateral filter
    • numpy – matrix operation
    • scipy.spatial.KDTree – kd-tree
  • Execute script
python2 src/panorama.py input_image/input_image.txt result/

Image Stitching

  • We have implemented a program to stitch the panorama image.

  • The source files contain:

    1. src/genImageList.py – Generate the image_list_file with image locations from a given directory.

    2. src/msop.py – Detect features and describe them of the input image by Multi-Scale Oriented Patch CVPR 2005.

    3. src/panorama.py – Construct the panorama images from an image list file with image locations and its focal length.

Contents

  1. Multi-Scale Oriented Patch

    1. MSOP Detection
    2. MSOP Description
    3. MSOP Match by k-d tree
    4. Adaptive Non-Maximal Suppression
    5. Sub-pixel Accuracy

  2. Panorama

    1. Image Match
      1. Affine Transform by pseudo inverse
      2. RANSAC
    2. Cylindrical Warping
    3. Detecting Connected Components
    4. Linear Blending
    5. De-drift

Procedures

  1. Warp the images to cylindrical coordinates

  1. Compute pairwise alignments by MSOP features

    1. Detect and describe MSOP features from warped images

    1. Match features between two images by k-d tree

  1. Construct panoramas

    1. Match images with affine transform by RANSAC from key points

      P=1−(1−pn)kwhere p=0.6,n=3,P=0.99that is k=18.92P=1−(1−pn)kwhere p=0.6,n=3,P=0.99that is k=18.92

    2. Recognize panoramas by connected components

    3. Blending linearly the warped images into panorama

    4. De-drift by compensate the average drifts per image

    5. Crop 10% parts of image height from top and bottom of the result image
